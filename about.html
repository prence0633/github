Project Overview

ERDucate is a web-based module designed to simplify the learning of Entity-Relationship Diagrams (ERD), databases, and SQL through interactive lessons, quizzes, and gamified activities.

Inspiration

The project was inspired by the struggles of students in comprehending abstract database concepts. ERDucate transforms these lessons into engaging, interactive experiences.

Objectives

  • Provide gamified lessons for ERD concepts.
  • Allow guided SQL practice.
  • Create quizzes and track progress.
  • Enhance confidence in database implementation.
