Introduction — Entities & Attributes

After class, John often misplaces his personal belongings, like his watch, bag, and notebook. Feeling frustrated, he struggles to keep track of them. Frankie shows him how to organize these items using basic database concepts: treating each item as an Entity and listing its details as Attributes. By assigning appropriate Data Types to each attribute (Text, Number, Date, Boolean) and arranging them in a simple table, John can clearly see and manage all his items. This exercise helps students define entities and attributes, recognize them in everyday situations, and build simple tables to organize information efficiently.

Summary — Entities & Attributes

Entity: An object you want to track (e.g., Watch, Bag)

Attribute: Details describing the entity (e.g., Color, Location)

Glossary — Key Terms

Entity
An object or thing you track in a database (e.g., Watch).
Attribute
A property or detail that describes an Entity (e.g., Color).
Data Type
Defines the kind of data an attribute holds (Text, Number, Date, Boolean).
