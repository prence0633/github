Introduction — SQL & Querying Databases

Frankie introduces SQL to help John retrieve data efficiently, while Mia observes. They explore basic SQL commands — SELECT, FROM, and WHERE — to query tables, filter results, and solve small tasks like finding lost items. By using SQL, John learns how to communicate with the database, quickly locate specific data, and apply queries to solve simple problems without manually searching through messy tables.

Glossary — Key Terms

SELECT
SQL command used to specify which columns to retrieve from a table.
FROM
Specifies which table to fetch data from.
WHERE
Condition to filter rows returned by the query.
Table
A collection of rows and columns that store data.
